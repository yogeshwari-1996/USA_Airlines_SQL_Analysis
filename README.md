# USA_Airlines_SQL_Analysis
This project analyzes US airline data using SQL to uncover insights on passenger trends, pricing strategies, and market competition.

## Tools Used
- PostgreSQL
- SQL
- Tableau (optional)

---

## Key Analysis Performed

- Top 10 busiest airline routes
- Fare trends over time
- Market share analysis of airlines
- Cheapest vs most expensive routes
- Fare per mile efficiency analysis
- Running total of passengers using window functions

## Project Structure

- `data/` → Dataset used  
- `sql/` → SQL scripts  
- `insights/` → Key findings  
- `dashboard/` → Visualization
